# 🪡Fashion Memory Game
A card-matching memory game with a fashion theme to match alike clothing pieces.
Built from scratch using vanilla HTML, CSS, and JavaScript 

## Play the game!
[Live demo ->](https://alfredthebuttler.github.io/fashion-memory-game)

## Fun Features

- 16 cards each with a different item of clothing, 8 matching fashion pairs shuffled on every game
- tracks moves played, matched pairs, and time elapsed
- Win message that reacts differently according to different scores
- Keyboard accessible - Tab to navigate, Enter or Space to flip
- Screen reader accessible - ARIA label announces each card's state (e.g. hidden, flipped, matched)

## Skills used/ Learned
- **DOM manipulation**
- **Game state management**
- **CSS custom properties**
- **CSS 3D transforms**
- **ARIA attributes**

## Built using 
- HTMLS
- CSS3
- Vanilla Javascript
